Server-side Slf4j and Log4j setup with some extra conveniences

### Getting Started

The library is available in the public Maven repository:

```
<dependency>
  <groupId>com.github.susom</groupId>
  <artifactId>server-logging</artifactId>
  <version>1.0</version>
</dependency>
```
